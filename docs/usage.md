# 使用教程

<div class="pinhaotu-highlight-box">

【重要】使用前请确保在科学上网（挂 VPN）环境下使用本插件，能正常浏览 Pinterest 网站即可正常使用本插件。

</div>

## 一、功能简介

Pin好图通过 Gemini API 识别你上传的参考图，自动生成搜索关键词，并在 Pinterest 上搜索相似图片，适合找参考、找风格、找配色。

## 二、API 密钥与隐私（首次使用必读）

- 需在 Google AI Studio 申请 Gemini API 密钥，在插件中保存后即可使用。  
  申请教程：打开 <https://aistudio.google.com/apikey> ，用 Google 账号登录后点击「Create API key」，选择或新建项目后即可生成密钥；将密钥复制到插件「API key」输入框并点击「保存」。
- 密钥仅保存在本机浏览器中，不会上传到 Pin好图 或其它服务器；图片经压缩后仅发送至 Google API 用于识图。

## 三、基本流程

1. 打开插件（弹窗或侧边栏）。
2. 上传 1～5 张参考图  
   - 未选图时：点击中央带「+」的方框，在弹窗中选择图片。  
   - 已选图后：预览会叠放显示；可点击每张左上角「×」删除单张；未满 5 张时仍可点「+」继续添加。  
   - 支持拖拽：将本地或网页上的图片拖入上传区即可添加。  
   - 支持粘贴：鼠标在上传区时按 Ctrl+V / Cmd+V 粘贴剪贴板中的图片。
3. （可选）填写「搜索范围提示」  
   - 如：汽车、logo、配色、氛围、扁平风 等，帮助 AI 缩小搜索范围，结果更贴题。
4. 点击「上传图片，在 Pinterest 搜相似图」  
   - 插件会压缩图片、调用 Gemini 解析并生成关键词，随后在 Pinterest 搜索并跳转结果页。

## 四、结果页使用

- 首次点击后会在当前标签页打开 Pinterest 搜索结果；建议在 Pinterest 站点下使用侧边栏，体验更顺。
- 若对关键词不满意：可修改「搜索范围提示」后再次点击同一按钮，会按新提示重新生成关键词并搜索。
- 按钮会变为「换一个关键词再试试」：可多次点击以切换多组备选关键词，无需重新上传图片。

## 五、小技巧

- 参考图尽量主体清晰、风格统一，更易得到好用关键词。
- 提示词写类别或风格（如「室内设计」「水彩」）比写过长句子更稳定。
- 侧边栏常驻在 Pinterest 页面，边看结果边换图、换提示词最方便。
- 侧边栏左上角「在弹窗中打开」可在新窗口中打开插件界面。
- 高级选项中可点击「检查更新」从 GitHub 检查新版本，有更新时可一键下载发布包。

## 六、常见问题

- 若 Pinterest 页面显示「抱歉！当前有太多用户使用我们的搜索功能。请稍后重试。」：这是 Pinterest 官方的限流或繁忙提示，并非插件故障。请稍等 1～2 分钟后再刷新页面重试，或尝试切换 VPN 线路；避免在短时间内连续多次点击「换一个关键词再试试」。插件会在检测到该提示时在页面顶部显示简短说明与「刷新」按钮。
